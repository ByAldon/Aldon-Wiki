# How to install the plugins

I will explain here how to install your plugins

***

1. Download the plugin from my github.
<img src="assets/exclusive-tutorials/wordpress/how-to-install-the-plugins/1.jpg">

2. Goto plugins inside your wordpress => Click "Add Plugin" => Click "Upload Plugin"
<img src="assets/exclusive-tutorials/wordpress/how-to-install-the-plugins/2.jpg">
<img src="assets/exclusive-tutorials/wordpress/how-to-install-the-plugins/3.jpg">

3. Pick the Plugin of your choise from your drive and hit "Install now"
<img src="assets/exclusive-tutorials/wordpress/how-to-install-the-plugins/4.jpg">

4. Once the plugin is uploaded you can either Activate the plugin or go back to the Plugin Installer.
<img src="assets/exclusive-tutorials/wordpress/how-to-install-the-plugins/5.jpg">


5. Your plugin installation is done! Have fun with your plugin!
